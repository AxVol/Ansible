## Описание playbooks
### ansible_service_user.yml
Создает в системе юзера с логином ansible, блокирует ему вход по паролю и грузит ssh ключи. 
#### Переменные
- password: - Пароль который будет установлен для пользователя ansible
- keys_path: -  пути к ssh ключам которые будут загружены на хост
### create_custom_user.yml
Создает в системе обычного пользователя без возможности использовать sudo. Состоит только в группах по умолчанию для хоста.
#### Переменные
- username: - Логин который нужно установить пользователю
- password: - Пароль который нужно установить пользователю
### create_custom_admin.yml
Создает в системе пользователя с админ правами. Playbook адаптирован под мандатный контроль astra linux устанавливая ему pdpl 63
#### Переменные
- username: - Логин который нужно установить пользователю
- password: - Пароль который нужно установить пользователю
