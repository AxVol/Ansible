Название роли
=========
## zabbix-agent

Данная роль предназначена для установки zabbix агента первой и второй версии на сервера, собранные из бинарей. 

Бинарники собраны для Debian-based на Ubuntu 18 и их работа проверена вплоть до Ubuntu 24, для RedHat-Based собраны на Alma linux 8, работа проверена вплоть до Alma linux 10. Так же успешная работа роли проверена на Astra Linux 1.7.X версиях.

Установка производиться в оффлайн режиме, дата сборки бинарей - `01.04.2026`

Переменные роли
--------------
### Прописывается в inventory
zabbix_version: - Cлужит указателем для того какую версию агента необходимо установить на данный хост

### Прописывается в defaults/main.yml
zabbix_ip: - В ней находится ip адресс заббикс сервера, который используется при генерации conf файла в j2 templates

Содержание files
--------------
agent1/ - Необходимые файлый для установки и работы zabbix agent 1
- zabbix-agent.service - systemd юнит с помощью которого запускается служба
- sbin/ - директория с бинарником для запуска агента

agent2/ - Необходимые файлы для установки и работы zabbix agent 2
- conf/ - Директория с предустановленными конфигруациоными плагинами zabbix agent 2
- deb/ - Установочные файлы и бинарники скомпелированные под Debian-based дистрибутивы
- yum/ - Установочные файлы и бинарники скомпелированные под Debian-based дистрибутивы
- zabbix-agent2.service - systemd юнит с помощью которого запускается служба

Содержание templates
--------------
- zabbix_1.conf.j2 - Конфигруационный файл службы, помещается в /etc/zabbix/
- zabbix_2.conf.j2 - Конфигруационный файл службы, помещается в /etc/zabbix/
