Название роли
=========

Устанавливает в систему docker в базовой конфигурации. Установка протестирована на Debian-based, RedHat-based и Astra Linux. Установка производиться в online режиме, offline не поддерживается.

Переменные роли
--------------

### Переменные в defaults/main.yml 
docker_user: - Логин пользователя под чьим именем ansible работает в системе чтобы добавить его в группу docker для будущего управления контейнерами
